1)假设我们的运算使用n位二进制数，那么这n位二进制数的模数为2n ,数α为一个用n位二进制表示的常数.  
α – x   
= 2n %2n + (α – x) % 2n  
        = (2n + (α – x)) % 2n  
        = (α + 2n – x) % 2n  
        = (α) % 2n + (2n – x) % 2n  
        = α + (2n – x)  
得出x com = 2n – x  

反码则简单的多，一个数的全部二进制位取反则得到其反码，由此可知，如果一个数加上它的反码，则此全部二进制位是满的，也就是全部是1，其值为2n-1 + 2n-2 + … + 22 + 21 + 20 = 2n – 1  
我们得出第二个结论  
xneg = 2n – 1 – x
综合结论一和二，我们可以做如下推导：  
xcom = 2n – x  
        = 2n – 1 – x + 1  
        = xneg + 1  
至此我们得出最终结论一个n位二进制数的补数等于其反码加一。
